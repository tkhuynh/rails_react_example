= Ruby on Rails + React Example

A CRUD example of Ruby on Rails app with ReactJS, this example was built following Piotr Jaworski's tutorial https://www.nopio.com/blog/react-rails-part-1-tutorial/

== Usage

* Fork and clone the repo

* Make sure PostgreSQL server is running

* In terminal: 
  
    rake db:create
    rake db:seed
    rails s
    open http://localhost:3000
